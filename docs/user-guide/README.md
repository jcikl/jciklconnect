# 用户指南

## 欢迎使用 JCI Kuala Lumpur 管理平台

本指南将帮助您快速上手并充分利用平台的各项功能。

## 🚀 快速开始

### 1. 账户注册和登录

#### 新用户注册
1. 访问平台首页
2. 点击"加入我们"按钮
3. 填写注册表单：
   - 姓名
   - 邮箱地址
   - 电话号码
   - 会员类型选择
4. 提交申请等待审核

#### 登录账户
1. 点击"登录"按钮
2. 输入邮箱和密码
3. 点击"登录"进入系统

### 2. 个人资料设置

登录后，首先完善您的个人资料：

1. 点击右上角的用户头像
2. 选择"个人资料"
3. 填写或更新以下信息：
   - 基本信息（姓名、电话、地址）
   - 职业信息（公司、职位）
   - 兴趣爱好
   - 技能专长
4. 上传个人头像
5. 保存更改

## 📋 主要功能使用指南

### 会员管理

#### 查看会员目录
1. 在侧边栏点击"会员"
2. 浏览会员列表
3. 使用搜索框查找特定会员
4. 点击会员名称查看详细资料

#### 更新个人信息
1. 进入"个人资料"页面
2. 编辑需要更新的信息
3. 点击"保存"确认更改

### 活动管理

#### 查看活动日历
1. 点击"活动"菜单
2. 选择日历视图查看所有活动
3. 点击活动查看详情

#### 报名参加活动
1. 在活动列表中找到感兴趣的活动
2. 点击"查看详情"
3. 点击"立即报名"
4. 填写报名信息
5. 确认报名

#### 取消活动报名
1. 进入"我的活动"页面
2. 找到已报名的活动
3. 点击"取消报名"
4. 确认取消

### 项目参与

#### 查看项目列表
1. 点击"项目"菜单
2. 浏览当前进行的项目
3. 使用筛选器按类别查看项目

#### 申请加入项目
1. 选择感兴趣的项目
2. 点击"申请加入"
3. 填写申请理由
4. 等待项目负责人审核

#### 项目进度跟踪
1. 进入"我的项目"页面
2. 查看参与项目的进度
3. 更新任务状态
4. 提交工作报告

### 积分和成就系统

#### 查看积分余额
1. 在仪表板上查看当前积分
2. 点击积分数字查看详细记录
3. 了解积分获取和使用规则

#### 查看徽章和成就
1. 进入"游戏化"页面
2. 查看已获得的徽章
3. 了解未完成的成就目标
4. 跟踪成就进度

#### 兑换积分奖励
1. 进入"会员福利"页面
2. 浏览可兑换的奖励
3. 选择想要的奖励
4. 使用积分兑换

### 财务信息

#### 查看会费状态
1. 进入"财务"页面
2. 查看会费缴纳状态
3. 了解到期时间
4. 在线缴纳会费

#### 查看交易记录
1. 在财务页面选择"交易记录"
2. 查看所有财务交易
3. 下载交易明细

### 治理参与

#### 参与投票
1. 进入"治理"页面
2. 查看当前进行的投票
3. 阅读投票详情
4. 选择投票选项
5. 提交投票

#### 查看选举信息
1. 在治理页面选择"选举"
2. 查看候选人信息
3. 了解选举规则
4. 参与投票

## 🔧 高级功能

1. 进入仪表板页面

3. 选择要添加的小部件

### 数据导出

#### 导出个人数据

2. 选择"数据管理"
3. 点击"导出数据"
4. 选择导出格式（PDF/Excel）
5. 下载文件

## 🆘 常见问题解决

### 登录问题
**问题：忘记密码**
1. 在登录页面点击"忘记密码"
2. 输入注册邮箱
3. 查收重置密码邮件
4. 按邮件指示重置密码

**问题：账户被锁定**
1. 联系管理员
2. 提供账户信息
3. 等待账户解锁

### 功能使用问题
**问题：无法报名活动**
- 检查活动是否还有名额
- 确认是否符合参与条件
- 检查网络连接

**问题：积分未正确显示**
- 刷新页面
- 检查积分获取条件
- 联系管理员核实

### 技术问题
**问题：页面加载缓慢**
- 检查网络连接
- 清除浏览器缓存
- 尝试使用其他浏览器

**问题：功能无法正常使用**
- 确保浏览器版本最新
- 启用 JavaScript
- 禁用广告拦截器
