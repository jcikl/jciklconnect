## 1. 付款申请列表页（组织财政 / 活动财政）

### 布局

```
┌─────────────────────────────────────────────────────────────────────────┐
│ [Logo] JCI LO 管理应用        [组织财政] [帮助] [用户]                    │
├─────────────────────────────────────────────────────────────────────────┤
│ 侧栏 │ 付款申请                                                          │
│      ├──────────────────────────────────────────────────────────────────┤
│ 导航 │ [搜索编号...] [状态 ▼] [活动 ▼]       共 12 条    [新建申请]      │
│      ├──────────────────────────────────────────────────────────────────┤
│      │ 编号               │ 申请人 │ 金额  │ 用途       │ 状态  │ 操作  │
│      ├──────────────────────────────────────────────────────────────────┤
│      │ ACT-2025-02-001-M  │ 小琳   │ ¥500  │ 迎新活动费 │ 待审  │ [审]  │
│      │ ACT-2025-02-002-M  │ 阿明   │ ¥200  │ 场地押金   │ 已批  │ [查]  │
│      │ ... ⚠️ 可能存在重复 │ ...   │ ...   │ ...        │ ...   │ ...   │
│      ├──────────────────────────────────────────────────────────────────┤
│      │                        [< 上一页] 1 2 3 [下一页 >]                 │
└─────────────────────────────────────────────────────────────────────────┘
```

### 关键元素

- **搜索**：编号搜索输入框，支持回车触发
- **筛选**：状态（全部/待审/已批/已拒）、活动（活动财政时仅本活动）
- **去重提示**：行内或筛选区显示「可能存在重复」
- **状态 Badge**：待审（琥珀）、已批（绿）、已拒（红），配文字不单靠颜色
- **分页/懒加载**：列表分页或懒加载，符合 NFR-P1

### 关键元素

- **带出字段**：只读或可覆盖；未选会员时允许手动填写（视产品策略）

## 3. 对账视图（按编号勾稽）

### 布局

```
┌─────────────────────────────────────────────────────────────────────────┐
│ 对账                                                                    │
├─────────────────────────────────────────────────────────────────────────┤
│ [输入参考编号...] [搜索]    时间范围 [开始] - [结束]                      │
├─────────────────────────────────────────────────────────────────────────┤
│ 银行流水                     │ 业务记录（付款申请/活动/会费）             │
│ ┌─────────────────────────┐ │ ┌─────────────────────────┐              │
│ │ 日期   金额  备注       │ │ │ ACT-2025-02-001 小琳    │              │
│ │ 02-15  ¥500  ACT-02-001 │ │ │ ¥500 迎新活动费  [匹配] │              │
│ │ 02-14  ¥200  ACT-02-002 │ │ │ ACT-2025-02-002 阿明    │              │
│ └─────────────────────────┘ │ │ ¥200 场地押金    [匹配] │              │
│                             │ └─────────────────────────┘              │
└─────────────────────────────────────────────────────────────────────────┘
```

### 关键元素

- **编号搜索**：输入编号后快速定位
- **流水与业务关联**：支持人工匹配、标记已到账
- **半自动建议**：同编号、同金额、时间窗内可建议匹配

## 5. 导航与页面关系

```
首页（按角色）
  ├── 付款申请列表 ──► 新建申请（表单，选会员即带出）
  │                └─► 申请详情 / 审核
  ├── 对账
  ├── 会员主档
  └── 帮助 / 新流程说明
```

---

## 6. 响应式断点

- **≥1024px**：侧栏 + 主内容，表格完整显示
- **768–1023px**：侧栏可折叠，表格可横向滚动
- **<768px**：列表卡片化，表单单列，触控目标 ≥44px
